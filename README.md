# My Portfolio Website

Ini adalah website portfolio pribadi saya yang berisi informasi tentang diri saya, skill yang saya pelajari, dan beberapa project yang sudah saya kerjakan.

Website ini dibuat untuk menunjukkan perjalanan saya dalam belajar dan berkembang di bidang teknologi, khususnya di area **Quality Assurance (QA)** dan software testing.

## Isi Website
- Tentang saya
- Skill & tools yang saya gunakan
- Beberapa project yang pernah saya kerjakan
- Kontak saya

## Dibuat dengan
- HTML
- CSS
- JavaScript
- Git & GitHub

## Cara membuka
Clone repository ini atau download project-nya, lalu buka file `portfolio.html` di browser.

```bash
git clone https://github.com/Fikriaz/PortfolioFikri.git
```
